DocuQuery Documentation

Your ultimate guide to utilizing the AI Data Extraction Specialist.

Overview

DocuQuery is an advanced Retrieval-Augmented Generation (RAG) tool powered by NVIDIA's Meta Llama 3.1. It allows you to upload heavy PDF documents (like financial reports, scientific papers, or tabular datasets) and instantly ask questions about them.

How to Use

  1. Upload: Navigate to the Application tab and drag your PDF into the upload zone. Wait for the processing to finish.
  2. Ask: Type questions into the chat box. Ask for specific numbers, metrics, or summaries.
  3. Analyze: The AI will extract the data from your document and present it cleanly using Markdown tables or bulleted lists.
  4. Reset: Use the Clear All Data button to wipe your local database and start fresh with a new document.

Privacy & Data

All embeddings are generated using NVIDIA's API and stored entirely locally in your own Chroma database folder. Your data is not permanently hosted on any external servers.
